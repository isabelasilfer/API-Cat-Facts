# 🐱 Cat Facts Traduzido

Aplicação que consome uma API pública para exibir fatos aleatórios sobre gatos e traduz automaticamente para o português.

## 🔧 Funcionalidades

- Consumo de API REST (catfact.ninja)
- Tradução automática do inglês para português
- Versão em Python
- Versão Web (HTML + JavaScript)
- Interface simples com imagem ilustrativa

## 🐍 Versão Python

Arquivo: `api.py`

- Utiliza requests para acessar a API
- Usa googletrans para tradução
- Exibe o fato original e traduzido no terminal

## 🌐 Versão Web

Arquivo: `index.html`

- Utiliza fetch para consumir a API
- Usa API pública de tradução (MyMemory)
- Exibe resultado dinamicamente na página

## 🚀 Tecnologias utilizadas

- Python
- requests
- googletrans
- HTML
- CSS
- JavaScript (fetch API)
